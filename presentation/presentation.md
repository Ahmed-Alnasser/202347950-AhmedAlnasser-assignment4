# Presentation

This folder contains the prepared materials for the Assignment 4 presentation.

## Files

- `slides.html` - editable presentation deck with print-friendly slide layout.
- `slides.pdf` - exported presentation deck for submission or screen sharing.
- `speaker-notes.md` - 5-7 minute talk track with timing and demo cues.
- `recording-checklist.md` - setup checklist for recording `demo-video.mp4`.

## Presentation Flow

1. Introduction and project motivation.
2. Live walkthrough of the portfolio application.
3. Technical architecture and AI integration.
4. Technical deep dive: solved challenges, remaining limitations, and future work.
5. Conclusion with outcomes and next steps.

## Recording Reminder

The final video file should be recorded manually after the latest app version is committed:

```bash
python3 -m http.server 8020
```

Open `http://localhost:8020`, record a 5-7 minute walkthrough using the script, and save the result as:

```text
presentation/demo-video.mp4
```
