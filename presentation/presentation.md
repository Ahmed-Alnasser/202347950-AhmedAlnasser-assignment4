# Presentation Materials

This folder contains the prepared presentation materials for SWE363 Assignment 4.

## Included Files

- `slides.html`: editable browser-based slide deck.
- `slides.pdf`: exported PDF version of the slide deck.
- `speaker-notes.md`: 5-7 minute speaker script with timing and demo cues.
- `recording-checklist.md`: checklist for recording the final demo video.

## Presentation Structure

The presentation follows the required assignment structure:

1. Introduction and project motivation.
2. Live demonstration of the portfolio application.
3. Technical architecture overview.
4. AI integration highlight.
5. Technical deep dive covering solved challenges, limitations, and future work.
6. Conclusion with outcomes and next steps.

## Demo Script Summary

Recommended demo path:

1. Open the deployed portfolio or local server.
2. Show the hero section and call-to-action buttons.
3. Demonstrate theme toggle and collapsible sections.
4. Open project links and the PenaltyOracle PDF.
5. Open achievement certificate modals.
6. Show contact form validation and AI summarizer behavior.
7. Conclude with deployment, documentation, and future improvements.

## Recording Instructions

Start the app locally:

```bash
python3 -m http.server 8020
```

Open:

```text
http://localhost:8020
```

Record a 5-7 minute walkthrough and save the final video as:

```text
presentation/demo-video.mp4
```
