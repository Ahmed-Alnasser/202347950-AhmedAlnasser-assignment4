Technical Architecture

Small Files, Clear Responsibilities

HTML Semantic sections, SEO metadata, contact form, modals.
CSS Base tokens, layout, sections, components, responsive rules.
JavaScript Theme, state, nav, contact, modals, scroll, typewriter.
Services EmailJS for messages and Transformers.js for local AI.

State Management

Local storage keeps theme and collapsed-section preferences persistent across reloads.

Compatibility

Progressive enhancements, reduced-motion support, lazy images, and graceful error messages keep the app usable if optional services fail.

Architecture is intentionally simple: no build step required. Slide 4/8

AI Integration Highlight

Local Message Summarizer

How It Works

  • Uses Transformers.js in the browser.
  • Loads a quantized summarization model on demand.
  • Lets the user replace a long message with the generated summary.

Responsible Design

  • Message content stays on the visitor's device.
  • The app warns before heavy local inference.
  • Failures show helpful messages instead of breaking the form.
The AI feature supports the user workflow instead of being decorative. Slide 5/8
