Technical Architecture

How The Complete Site Is Organized

CSS Theme tokens, layout, sections, components, responsive rules, and print styles.
JavaScript Theme, state, navigation, contact, modals, scrolling, typewriter, and AI.

Stateful Experience

Theme and collapsed-section preferences persist across reloads, making the site feel like an application instead of a static page.

Production Readiness

Reduced-motion support, lazy images, graceful errors, accessibility attributes, and deployment automation support real users.

The architecture stays simple enough to maintain while still supporting rich features. Slide 4/8

AI Integration Highlight

AI As Part Of The Contact Workflow

How It Works

  • Uses Transformers.js in the browser.
  • Loads a quantized summarization model on demand.
  • Helps visitors shorten long contact messages before sending.

Responsible Design

  • Message content stays on the visitor's device.
  • The app warns before heavy local inference.
  • The form still works if the AI model or network fails.
The AI feature improves a real workflow while preserving user control and privacy. Slide 5/8
